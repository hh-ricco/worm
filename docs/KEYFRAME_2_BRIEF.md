# 第二张关键帧 Brief：章节 1 全屏状态

**目标**：画出整个游戏屏幕在"章节 1（观察）"状态下的样子。这一张图画完，整个作品的视觉骨架就立住了——之后所有其他状态都是这张图的"状态变化"。

**不要做的事**：
- ❌ 不要画其他章节状态（食物滴落、危险区出现等）—— 那是之后的关键帧
- ❌ 不要画动画帧 —— 静态构图即可
- ❌ 不要画 AI 揭示页 —— 那是另一张关键帧
- ❌ 不要做移动端布局 —— 桌面优先，移动版后续适配

---

## 1. 画布与基础

- **画布尺寸**：1920 × 1080 px（标准 16:9）
- **工作工具**：Figma（首选）或 Illustrator
- **导出预期**：SVG 优先，关键元素分组命名（便于代码引用）
- **页面背景**：暖灰白底色（建议 `#FAF7F0` 或 `#F5F3ED`）—— 不是纯白，避免刺眼

## 5. 各分区详细 spec

### 5.1 顶栏

**左侧**：章节进度条
```
章节 1 / 7  ●  ○  ○  ○  ○  ○  ○    观察
```
- 7 个圆点，当前章节是实心彩色（用 AVB 蓝灰），其他空心
- 悬停每个点显示该章节名（虽然这张图不需要画悬停态）

**右侧**：设置齿轮图标 + 音效开关图标（一个小喇叭）

### 5.2 培养皿区域（占据画面最大权重）

- **形状**：完美圆形，居中
- **大小**：直径约 850-900 px（占据左侧区域）
- **外环**：深灰描边 4-5 px，模拟玻璃皿边缘
- **内部**：极浅米色，可有非常微妙的同心圆暗示（透明度 5-10%）
- **底部小阴影**：椭圆形软阴影，10-15% 不透明度，给整个皿子轻微"摆在桌上"的感觉
- **线虫**：在皿中心略偏处，方向自由，使用你已确认的视觉风格
- **不要画**：食物、危险区、温度热力图、光斑——这些是后续章节解锁后才出现的

可选细节（看你时间）：
- 培养皿底部的纹理印记（比如品牌"C. elegans Lab"的小字标签，致敬感）
- 极淡的网格背景在皿外（强化"实验室"氛围）

### 5.3 工具盘（章节 1 全部灰态）

四个工具按钮，2×2 网格或 1×4 横排：

```
┌─────────┐ ┌─────────┐
│  🍃 🔒  │ │  ⚠️ 🔒  │
│         │ │         │
│  食物    │ │  危险    │
└─────────┘ └─────────┘
┌─────────┐ ┌─────────┐
│  🌡️ 🔒  │ │  💡 🔒  │
│         │ │         │
│  温度    │ │  光照    │
└─────────┘ └─────────┘
```

每个按钮：
- 卡片：白底 + 2px 描边 + 8px 圆角
- 中央：工具图标（线条风，与整体描边风一致）
- 下方：工具名（中文 16px）
- 右上角：小锁图标（2px 线条）
- 整体灰态化（饱和度 0%，不透明度 65%）

**图标设计建议**（每个简单几何）：
- 🍃 食物：一个小水滴或一颗豆子的形状
- ⚠️ 危险：三角警示或多个尖刺图案
- 🌡️ 温度：一个温度计或波浪线
- 💡 光照：一个灯泡或太阳放射状

不需要拘泥于 emoji，重点是**风格统一**——都是描边线稿，跟线虫一个调调。

### 5.4 示波器面板（核心视觉锚点）

整个面板深底，5 条频道堆叠：

```
┌──────────────────────────────────────────────────┐
│  神经元活动（rate coding）                        │
│ ─────────────────────────────────────────────    │
│                                                   │
│  AVB  运动指令          ─┃──┃──┃──┃──┃──        │ ← 活跃
│                                                   │
│  AWC  食物气味  🔒     ──────────────────         │ ← 灰态
│                                                   │
│  ASH  伤害感受  🔒     ──────────────────         │
│                                                   │
│  AFD  温度感受  🔒     ──────────────────         │
│                                                   │
│  ASJ  光感受    🔒     ──────────────────         │
│                                                   │
└──────────────────────────────────────────────────┘
```

每条频道：
- 高度约 90-100 px
- 左侧 ~140 px：标签区
  - 上行：神经元学名（mono 字体，激活时浅灰白）
  - 下行：日常名（中文 sans，稍小）
  - 灰态：整个标签变暗灰 + 右侧加锁
- 右侧：滚动轨迹区
  - 黑/深底
  - 极淡水平网格线（横向几条）
  - 尖峰为竖线（活跃时）/ 一条平直线（灰态）
  - **AVB 在静态截图里可以画 5-8 个分布不均的尖峰**（模拟泊松噪声），高度 80-90% 频道高度
  - 其他 4 条画一条 1px 的平直线，明显"死寂"

面板顶部小标题："神经元活动（rate coding）" + 一个小问号图标（提示"什么是 rate coding？"）

### 5.5 旁白条

- 浅色横条（可比页面背景略暗，比如 `#F0EDE5`）
- 居中文本，左侧可以有一个小的 💬 图标
- 章节 1 文案（建议）：

> "这是一只线虫。它只有 302 个神经元——比你手机里的晶体管少一万亿倍。
> 先看着它走两分钟。"

- 字号 18-20 px，行高 1.6
- 文本两侧留白 200+ px

## 6. 章节 1 状态特别说明

这一帧的核心信息是 **"几乎什么都没解锁"**：

✅ 必须传达的状态：
- 培养皿是空的（除了线虫）
- 4 个工具全灰 + 锁
- 5 条神经元频道里只有 AVB 活跃
- 旁白告诉玩家"先看一会儿"

❌ 不要画：
- 任何刺激物（食物滴、危险区）
- 任何彩色工具
- 任何被解锁的频道（除了 AVB）
- "下一章" 按钮（章节 1 还没完成）
